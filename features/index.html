Everything You Need for Agentic Commerce

From UCP discovery and Pulse Concierge to the Abilities API and GEO Suite—CognitiveCart makes your WooCommerce store agent-ready.

Free

Pulse Concierge

Detect high-intent visitors and show them helpful nudges. Triggers are grouped by customer intent; when 2–4 triggers in a group fire (score ≥ 30), the AI gets that group's context and generates a relevant, sales-focused nudge.

  • Intent triggers: semantic hover, variant cycling, coupon hover, cart stall, scroll depth, product views, and more
  • Optional AI chat widget and AI-generated nudges (requires Worker URL)
  • Context-aware intent classification; machine commands (add to cart, checkout, apply coupon, etc.)

Trigger groups

Considering product, cart / value-seeking, near purchase—each with group-specific AI prompts for sales-focused nudges.

Registered abilities

Store capabilities exposed for AI agent discovery via WordPress Abilities API and UCP manifest.

  • add-to-cart, empty-cart, get-checkout-url
  • create-checkout-session (3rd party agents)
  • remove-item, update-item, apply-coupon, remove-coupon
  • get-chat-context, list-products, get-cart-summary
Free

Abilities API

Register store capabilities with the WordPress Abilities API so AI agents (and WooCommerce MCP) can discover and invoke cart and checkout actions. Add to cart, get checkout URL, create checkout session for third-party agents, list products, get cart summary, and more—all with defined input schemas.

Free

UCP / ACP Gateway

Your UCP manifest is served automatically at /.well-known/ucp. No configuration required. AI crawlers discover your store and see capabilities, abilities endpoint, and (when Stripe Connect is configured) checkout create-session endpoint and complete URL template.

  • Third-party AI checkout: discover → GET products → POST checkout/create-session → Stripe Checkout
  • Source attribution: agent:Claude, agent:ChatGPT for order attribution

When Worker URL is set, manifest is hosted in KV and served from the edge. Otherwise served from WordPress.

Emergency control

Kill-Switch

Emergency toggle to suspend all autonomous agent activity. When active, Pulse and agent-facing features are suspended so you stay in control of when AI can interact with your store.
